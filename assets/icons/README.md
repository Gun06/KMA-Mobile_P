# Icons 폴더

이 폴더는 앱에서 사용하는 아이콘 파일들을 저장합니다.

## 폴더 구조

```
icons/
├── app/           # 앱 아이콘
├── ui/            # UI 아이콘
├── weather/       # 날씨 관련 아이콘
├── navigation/    # 네비게이션 아이콘
└── social/        # 소셜 아이콘
```

## 지원 형식

- PNG (권장)
- SVG (벡터 아이콘)
- ICO (Windows용)

## 크기 규칙

- 24x24px (기본)
- 32x32px (중간)
- 48x48px (큰)
- 64x64px (매우 큰)

## 네이밍 규칙

- 소문자와 언더스코어 사용
- 예: `home_icon.png`, `weather_sunny.svg`
